# Security policy

## Supported versions

Security fixes are applied to `main` and the latest release line. Older releases
are unsupported unless listed here.

## Reporting a vulnerability

Do not open a public issue. Use GitHub private vulnerability reporting:

<https://github.com/struktly/struktly/security/advisories/new>

Include the affected version or commit, impact, and a minimal reproduction. Do
not include credentials or unrelated private source.
